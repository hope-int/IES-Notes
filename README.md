## üöÄ Getting Started

### Prerequisites
- Node.js (v18+)
- A Supabase Project

### Installation

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/your-username/ies-notes-clay.git
    cd ies-notes-clay
    ```

2.  **Install Magic**:
    ```bash
    npm install
    ```

3.  **Configure Env**:
    Create a `.env` file:
    ```env
    VITE_SUPABASE_URL=your_supabase_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    VITE_OPENROUTER_API_KEY=your_openrouter_key
    ```

4.  **Launch**:
    ```bash
    npm run dev
    ```
